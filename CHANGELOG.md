## [0.1.0] - 2026-03-16

### Added
- Initial release with complete desktop POS functionality
- JWT-based authentication with role-based access control
- Dashboard with real-time metrics and quick actions
- Point of Sale checkout with cart management
- Product catalog with hierarchical categories
- Inventory management with stock tracking
- Customer database with purchase history
- Supplier management
- Sales history with refund support
- Reports with date filtering and analytics
- Purchase orders workflow
- Cash shift management
- Settings for store configuration
- User management interface
- Material Design 3 UI with custom theme
- Debug mode for development
- CI/CD pipeline with GitHub Actions
- Code quality tools (Spotless, Detekt, Kover)
- Database migrations system
- Comprehensive documentation

### Technical Stack
- Kotlin 2.3.10
- Compose Multiplatform 1.10.2
- Ktor 3.4.1
- Supabase PostgreSQL
- Koin 4.1.1
- Material 3

### Platforms
- Desktop (JVM) - Production Ready ✅
- Android - In Development 🚧
- iOS - In Development 🚧
- Web - In Development 🚧
